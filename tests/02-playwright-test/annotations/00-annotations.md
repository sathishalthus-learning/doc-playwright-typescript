### Annotations
```
    - test.only - runs 'only' marked tests 
    - test.skip - runs tests other than marked by 'skip'
    - test.skip(condition) - skips tests based on condition
    - test.describe - gives logical scope to tests
    - tag tests - adds details to test, can be added to describe/test functions
    - conditional skip group - sames as test.skip but inside test.describe
    - test.fixme - skips  hooks
    - runtime-annotations - adds annotation at runtime
```
